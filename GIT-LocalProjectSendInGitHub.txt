1. Создать репозиторий на удаленном сервере (GitHub и т.п.)
	Не создавать доп. файлы во избежании ошибки.
2.  В Git Bash переходим в нужную директорию.
	[cd /c/data/project/]
3. Определяем текущую директорию, как директорию репозитория:
	[git init]
4. Добавьте файлы в ваш новый локальный репозиторий, чтобы добавить файлы в первый коммит:
	[git add]
5. Коммит файлов, который вы добавили в коммит в ваш локальный репозиторий:
	git commit -m "The first commit"
6. Скопируйте ссылку на ваш репозиторий.
7. В командной строке добавьте ссылку на удаленный репозиторий:
	[git remote -v] (проверка удаленного адреса)
	[git remote add origin ссылка на удаленный репозиторий] (указание на новый удаленный репозиторий).
8. Отправка изменений локального репозитория в Git:
	[git push origin main(или 'master')]
	лучше так --> git push --set-upstream origin master
